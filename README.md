# **Scientific Calculator**

A Python Tkinter GUI calculator supporting basic arithmetic, scientific, and combinatorial operations in a clean, user-friendly interface.

---

## **Features**
- Basic: +, -, *, /  
- Scientific: sin, cos, tan, log, ln, √, powers (^), factorial (!)  
- Constants: π, e  
- Combinatorial: nCr, nPr  
- AC (clear all), DEL (delete last), error handling  

---

## **Technologies Used**
- Python 3.x  
- Tkinter for GUI  
- Math module for calculations  
- Regex for input parsing  

---

## **Development Approach **
- GUI designed with **Tkinter** for a professional layout  
- Buttons and input parsed using **regex** and **math module**  
- Handles errors gracefully and removes extra white space for clarity
